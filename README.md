## Hardware

![Hardware](https://i.imgur.com/iOjrwyD.jpeg)

- `Gigabyte B450 AORUS ELITE V2`:
    - CPU: `AMD Ryzen 5 2600`
    - RAM: `32GB`
    - SSD: `256GB`
    - 4 × HDD: `4TB`
- Backup Server: `Raspberry Pi 4`
    - PoE HAT
    - HDD: `16TB`
- NETGEAR `GS308EP` switch:
    - Ports: `8 PoE`
    - Speed: `1000Mbps`
